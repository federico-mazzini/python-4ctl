# Python 4CTL

Materiali, esempi ed esercizi per introdurre Python in 4CTL e arrivare alla realizzazione di mini applicazioni collegate a MySQL.

## Obiettivo del repository

Questo repository raccoglie:

- spiegazioni sintetiche in Markdown
- esempi Python già svolti
- esercizi da completare

Il progetto finale verrà sviluppato in un repository separato, in modo da mantenere questo repo pulito e focalizzato sul percorso di apprendimento.

## Struttura del repository

- `docs/` — appunti e spiegazioni sintetiche
- `esempi/` — un file per ogni costrutto o mini esempio
- `esercizi/` — un file per ogni esercizio, con la consegna nei commenti iniziali

## Ordine consigliato

1. Python base
2. Funzioni, liste e piccole applicazioni
3. Collegamento Python-MySQL
4. Distinzione tra logica applicativa e database
5. Progetto finale (in un repository separato)

## Come lavorare

1. Fare il fork del repository
2. Clonare il proprio fork in locale
3. Completare gli esercizi nei file indicati
4. Salvare il lavoro con commit chiari e frequenti

## Convenzioni consigliate

- usare nomi di file in minuscolo con underscore
- scegliere nomi di variabili e funzioni chiari
- spezzare il codice in funzioni quando il programma cresce
- usare query parametrizzate quando si lavora con il database
- separare il più possibile input, controlli logici e operazioni sul database

## Note

Questo repository non contiene la soluzione completa del progetto finale.
Le soluzioni degli esercizi possono essere condivise dal docente in un secondo momento, quando utile.
